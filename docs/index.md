# LAAL VLA Study

Vision-Language-Action(VLA) 모델 논문을 함께 읽고 정리하는 스터디 공간입니다.
로봇 조작(robot manipulation)에 언어와 비전을 결합하는 최신 연구 흐름을 추적하고,
매주 논문을 발표 및 토론하며 내용을 이 사이트에 아카이빙합니다.

## 스터디 목표

- VLA(Vision-Language-Action) 계열 모델의 핵심 아이디어와 흐름을 이해한다
- 매주 1편 이상의 논문을 정리하고 발표한다
- 발표 자료와 토론 내용을 기록으로 남겨 누구나 참고할 수 있도록 한다

## 진행 방식

1. 매주 발표자 1명이 논문을 선정하고 발표 자료를 준비합니다.
2. [스터디 계획](schedule.md) 페이지에 일정과 발표자를 기록합니다.
3. 발표 후 [논문 리뷰](papers/index.md)에 정리본을 추가합니다. [리뷰 템플릿](papers/_template.md)을 복사해서 작성하세요.

# 스터디 계획

!!! tip "사용 방법"
    아래 표의 날짜·발표자·상태를 실제 일정에 맞게 수정하세요.
    논문 목록은 VLA(Vision-Language-Action) 분야의 대표적인 논문들로 구성한 **예시 커리큘럼**이니
    스터디 방향에 맞게 자유롭게 추가/삭제/순서 변경하세요.

## 진행 규칙

- 주 1회, 매주 한 편의 논문을 정리하여 발표합니다.
- 발표자는 발표 전까지 [논문 리뷰](papers/index.md)에 정리본을 업로드합니다.
- 상태는 `예정` → `완료` 로 갱신합니다.
